# Path


Symbols | Description
------------|-----
M | Move cursor to specific coordinates
L | draw from the current position to destination position


## Examples:

```html
<path d="M100,100 L150,150"
      style="stroke: #ff0000;
      		 stroke-width: 3;
      		 fill: none;
      		"
/>
```
