## Generate latex into svg

First generate `latex` to `pdf`:

```bash
pdflatex example2.tex 
```
next convert `pdf` to into `svg`

```bash
pdf2svg example2.pdf example2.svg
```

**Pay attention**: the circles are are actually **paths**.
Not so good
